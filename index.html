HTML

Le HTML (HyperText Markup Language) est un langage de balisage utilisé pour créer et structurer des pages web. Il permet de définir la mise en page et le contenu d'une page, en organisant des éléments comme du texte, des images, des liens, des tableaux, des formulaires, etc..

Rôle et Apport dans mes Projets :

HTML est le langage de base pour structurer le contenu de toutes les pages web. Dans mes projets de classe, il a permis de concevoir la structure sémantique des sites, en organisant les éléments de manière logique et accessible. Grâce à HTML, j’ai pu : Créer des pages web claires et structurées, en utilisant les balises adaptées (titres, paragraphes, listes, etc.). Optimiser le référencement naturel (SEO) de mes projets en respectant les bonnes pratiques, telles que l’utilisation de balises de titre et de descriptions. Intégrer des médias (images, vidéos, sons) et des formulaires pour interagir avec les utilisateurs. Projets Marquants : Site de présentation d’entreprise : Structure complète avec des sections bien définies (Accueil, Services, Contact). Portfolio interactif : Utilisation de balises sémantiques pour faciliter l’accessibilité et l’indexation par les moteurs de recherche.

CSS

Le CSS, ou Cascading Style Sheets, est un langage utilisé pour décrire la présentation d'un document écrit en HTML ou XML. Il permet de contrôler l'apparence des éléments sur une page web, comme les couleurs, les polices, les espacements, et bien plus encore. Grâce au CSS, les développeurs peuvent créer des designs attrayants et responsives, rendant les sites web plus agréables à utiliser.

Rôle et Apport dans mes Projets :

CSS est le langage qui donne vie aux pages HTML en définissant leur style visuel. Il permet de contrôler la présentation de chaque élément et d’offrir une expérience utilisateur agréable. Dans mes projets, CSS a été essentiel pour : Concevoir des interfaces utilisateur attractives et responsive, adaptées à tous les types d’appareils (smartphones, tablettes, ordinateurs). Créer des animations et des effets visuels pour rendre les interactions plus dynamiques et engageantes. Personnaliser le design des éléments de mes projets, comme les boutons, les menus de navigation et les formulaires. Projets Marquants : Site de e-commerce : Conception d’un thème graphique personnalisé et adaptation des pages pour les appareils mobiles. Jeu de mémoire en ligne : Utilisation des animations CSS pour fluidifier les transitions et les effets de survol.

JavaScript

Le JavaScript est un langage de programmation principalement utilisé pour créer des pages web interactives. Il permet d'ajouter des fonctionnalités dynamiques aux sites, comme des animations, des formulaires interactifs, et bien plus encore. En plus d'être utilisé côté client (dans le navigateur), JavaScript peut également être utilisé côté serveur grâce à des environnements comme Node.js. C'est un langage très populaire et essentiel pour le développement web moderne.

PHP

Dynamiser et interagir avec les bases de données


Rôle et Apport dans mes Projets

PHP est un langage de script côté serveur, idéal pour la création de sites web dynamiques et pour l’interaction avec des bases de données. Au sein de mes projets, PHP a joué un rôle clé pour : Créer des systèmes de gestion de contenu (CMS) personnalisés, permettant la mise à jour facile des données depuis une interface utilisateur. Concevoir des formulaires de contact et des systèmes d’inscription, avec une gestion sécurisée des informations saisies par les utilisateurs. Intégrer et manipuler des bases de données MySQL pour stocker et afficher les données dynamiquement sur le site. Projets Marquants : Système de réservation en ligne : Développement de la logique de gestion des réservations avec interaction en temps réel avec une base de données. Blog personnel : Création d’un espace d’administration permettant de gérer les articles, les commentaires et les utilisateurs.
